# Reglas de diseño (home)

Objetivo:
-interfaz: moderna, premium y muy limpia.
- prioridad: legibilidad + jerarquía visual + sensación "producto de pago"
- una sola página (sin navegación compleja. Todo debe sentirse rápido).

Estilo general
-Tono claro por defecto. fondo gris con variaciones de blanco entre secciones
- toques de color solo para estados/acciones (no llenar de colores).
- Componentes consistentes: mismos radios, sombras y espacioados n toda la web.
Tipografia: 
- Fuente: inter (o sistema si no está disponible)
Titulos: 20-24px, BOLD
Subtitulos/labels: 12-14px, médium, con opacidad ligera.
-Texto normal: 14-16px, regular.
números: grandes, claros, con separados y símbolos $.

Layout y espaciados.
-Grid de 12 columnas (responsive).
-Espaciado base 8px (8/16/24/32)
-Contenedor max width: 1200px, centrado.
-Separación entre secciones: 24-32px
-mucho aire: evita bloques apretados.

tarjetas (cards)
-radio: 14-16px
-fondo de tarjeta: un tono más claro que el fondo.
Borde sutil (1px) o sombra muy suave, pro no ambos fuertes.
-padding interno: 16-20px
títulos de tarjeta abajo a la izquierda, acciones (-) arriba a la derecha su aplica.

colores:
-fondo principal: #F2F2F2
-superficie cards: #D9D9DA
- texto principal: #02080B
-texto secundario: #808385
-color de acento (botones/highlights): #249BE5
-estados: 
 -Positivo: verde suave
 -Negativo: rojo suave
 -Aviso: ambar suave

Botones e imputs
- botón primario: relleno con color de acento, alto 40-44 px, radio 12px
botón secundario: borde sutil o fondo ligeramente distinto, sin ruido. 
- inputs: oscuros, borde sutil, focus visible con acento.
-todo de ser "teclado-friendly"tab order lógico, enter para guardar, esc para cerrar.
